NACIFAŠIZEM

Vzroki za uspeh nacifašistične ideologije:
    - Nezadovoljstvo po 1. svetovni vojni:
        Nemčija je bila prisiljena sprejeti pogoje ostalih držav
        "Vittoria mutilata" - Italijani niso dobili vseh ozemlji, ki so hoteli zaradi Londonskega pakta in Willsonovih točk
    - Nasilje:
        Ljudje so se borili, klali, tepli in niso bili vseli
        Vojaki so bili navajeni nasilja --> Dosti so ostali brez službe
        Psihološko so vojaki zelo težko nehali streljati
        Ti vojaki največ sprejmejo te ideologije, ker so bili že navajeni
    - Anti-komunizem:
        Strah pred rdečim valom, ki se širi z vzhoda Evrope
        Še posebej v severu Italije se pojavlja Socializem
        Fašizem in Nacizem so Anti-komunistične stranke
        So proti liberlanim strankam, ker so takrat liberlane stranke začele vojno
    - Renovacija Evrope:
        Nudijo neko spremembo/novost
